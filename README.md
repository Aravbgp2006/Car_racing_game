# 🚗 Car Racing Game

A simple and fun **2D Car Racing Game** built using **HTML, CSS, and JavaScript**. The player controls a car, avoids incoming traffic, and tries to achieve the highest possible score.

---

## 📌 Features

- 🚘 Smooth car movement
- 🛣️ Endless racing track
- 🚗 Random enemy cars
- 🎮 Keyboard controls
- 📊 Live score counter
- 💥 Collision detection
- 🔄 Restart game after crash
- 📱 Responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎮 Controls

| Key | Action |
|------|---------|
| ⬅️ Left Arrow | Move Left |
| ➡️ Right Arrow | Move Right |
| ↑ Up Arrow | Accelerate |
| ↓ Down Arrow | Slow Down |

## 🚀 How to Run

1. Download or clone the repository.

```bash
git clone https://github.com/your-username/car-racing-game.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation required.
